Image Placement Guide

This page is for your reference only — not linked publicly. Use it to track which images go where.

To add an image, upload the file to the images/ folder, then replace the placeholder text with an <img> tag as shown below.

How to add an image

/* Replace a placeholder div like this: */

<div class="blog-card__img-ph">[ Athens Greece Photo ]</div>


/* With an actual img tag like this: */

<img class="blog-card__img" src="../images/athens-blog-cover.jpg" alt="Athens Greece in winter" loading="lazy">
